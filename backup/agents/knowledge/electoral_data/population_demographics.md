## 戦略示唆

### 人口ボリューム上位5町（=票田として大きい）
1. 川崎町 2,565人 → 投票区16
2. 川合町 2,480人 → 投票区11
3. みずきが丘 2,455人 → 投票区14
4. 能褒野町 2,305人 → 投票区15
5. みずほ台 1,875人 → 投票区13

### 投票率と人口のクロス分析
- **大票田×低投票率**=活性化余地大: みずきが丘（33.12%）、能褒野（31.68%）、みずほ台（32.93%）→ **新興住宅地の若年層投票誘発が鍵**
- **大票田×高投票率**=固い票田: なし（大票田はすべて30%台）
- **小票田×高投票率**=固いコア層: 下庄（草川地元）、辺法寺、野登、池山

### 草川氏の年齢戦略
草川氏は34歳（2022年時点）。同世代（30代）は2,464+2,885=5,349人=人口の11.0%だが、投票率は低い。
**戦略**: 子育て世代（30-44歳：8,615人）への訴求が、競合の少ない政策ニッチとなる可能性。
