# 草川たくや｜暮らし・福祉アーカイブ
最終更新: 2026-04-25（議会会議録 平成30年〜令和7年 全20セッション抽出済）

## エグゼクティブサマリー

- **主要主張**（5本）
  1. 救急搬送4割が市外＝亀山市民の命のインフラを守る
  2. 「誰ひとり取り残さない」防災（特に要配慮者の個別避難計画）
  3. 健康都市政策＝予防・聴覚・口腔・栄養の総合戦略
  4. 介護予防・自立支援は「協働の見える化」で
  5. 物価高騰時代の家計支援を市政で実装

- **一貫性スコア**: A
- **特徴**: 救急医療・防災では「数字で危機感を語る」スタイル（4割搬送・90.4%応答・震度6弱想定等）。voice-dna完全整合
- **未触れ重要領域**: 障害者就労、生活困窮者自立支援、認知症対策、ひきこもり支援等

## 3. 時系列進化（議会一般質問テーマから抽出）

### 2018-2019（初当選期・1期序盤）
- **健康都市政策**を初質問から取り上げ（介護予防・自立支援）
- 防災（地区防災計画）、エネルギー（分散型）でリスク管理視点

### 2020-2021（コロナ期）
- **コロナ・物価・教育・医療**の連動で家計支援・生活防衛
- 三重とこわか国体・大会、PCR検査・ワクチン接種運用

### 2022-2023（1期終盤〜2期始動）
- 物価高、家計支援、危険木、道路安全等の生活基盤論点
- 環境基本計画、カーボンニュートラル

### 2024-2025（2期中盤〜現在）
- 救急医療4割市外、母子医療、ワクチン拡充、口腔ケア等の**医療体制深掘り**
- 自動車避難容認、要配慮者個別避難計画等の**防災運用論点**
- 高齢者の社会参加（eスポーツ）、亀山茶機能性等の**新型福祉**

**進化の特徴**: 「制度の運用・データ要求」軸が一貫。ライフライン（医療・防災・水）と新時代福祉（eスポーツ・機能性食品）を両軸で展開。

## 6. policy-synthesizer向けフィード（応援カード政策3本柱用）

### 暮らし・福祉 — 推奨 短期/中期/長期 政策セット

#### 短期施策の種（〜6ヶ月、議会質問・運用改善レベル）
1. **救急搬送データの公開要求**: 過去5年の市外搬送率・平均搬送時間を6月議会で要求
2. **医師確保策の進捗確認**: 亀山市立医療センターヒアリング・寄付講座等の検討
3. **要配慮者個別避難計画の策定率公表**: 危機管理課への質問
4. **オンライン医療相談導入**: 産婦人科・小児科 全国230自治体実績ベース

#### 中期施策の種（6ヶ月〜2年、計画化レベル）
1. **地域医療構想（三重県）への市要望**: 二次救急＋高度医療の役割分担明確化
2. **要配慮者個別避難計画の100%策定**: 自主防災組織との連携事業
3. **高齢者の聞こえ支援条例**: 補聴器助成と健診連動
4. **ママ・サポート119**: 出産予定日事前登録制度
5. **eスポーツ条例・推進拠点**: 高齢者参加と若者地域定着の二重戦略

#### 長期施策の種（2年〜、構造改革レベル）
1. **地域包括ケアシステムの本格運用**: 認知症・在宅医療・介護の三位一体
2. **障害者就労支援の充実**: 自治体雇用＋民間連携
3. **生活困窮者自立支援の強化**: 包括相談・住居・学習・就労の4機能統合
4. **インクルーシブ社会条例**: LGBTQ・障害・外国ルーツ含む
