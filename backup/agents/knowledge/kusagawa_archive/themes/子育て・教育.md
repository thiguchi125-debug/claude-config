## 3. 時系列進化

### 2018-2019（初当選期）
- 「待機児童ゼロ」「保育環境改善」を初期公約に
- 「あんぜんの丘」こども園竣工式出席等、現場踏査

### 2020-2024（1期〜2期前半）
- 「行かない窓口」の前段としての行政DX論
- 五つの無料化（給食費・学用品費・副教材費・修学旅行費・通学費）の段階的推進

### 2025-2026（2期中盤・現在）
- **質的転換: 「制度を作る」→「制度の運用を改善する」フェーズへ**
- EBPM（数字・他自治体比較）を徹底
- 市民相談を起点にしたミクロな運用改善提案が増加
- 子どもの権利・プライバシーへの言及（着替問題）

**進化の特徴**: 国制度の実装 → 亀山独自の運用設計 → 市民の声からの微修正、と上流下流を行き来する論理回路が確立。

## 6. policy-synthesizer向けフィード（応援カード政策3本柱用）

### 子育て・教育 — 推奨 短期/中期/長期 政策セット

#### 短期施策の種（〜6ヶ月、議会質問・運用改善レベル）
1. **誰でも通園制度の運用見直し**: 慣らし保育期間を月10時間枠から分離、給食なし時間設定の見直し（6月定例会で提起）
2. **産後ケア利用券の対面交付**: 母子手帳交付時・こんにちは赤ちゃん訪問でのその場交付化
3. **小学校着替環境実態調査**: 教育委員会への照会＋カーテン整備計画策定
4. **子育てオンライン申請の優先実装**: 保育園申込・現況届・支給認定・妊娠届出の4本柱

#### 中期施策の種（6ヶ月〜2年、計画化レベル）
1. **オンライン医療相談導入**: 産婦人科・小児科24/365、年300万円予算化
2. **ママ・サポート119**: 消防指令センター連携、出産予定日事前登録制度
3. **学童保育の量・質改善計画**: 待機ゼロ＋障害児受入＋指導員処遇改善
4. **就学前施設再編の透明性確保**: 待機児童カウント方法の市民可視化、駐車場・給食室課題の段階的解決
5. **行政DX進捗100%目標**: 県内他市同水準への引き上げ計画

#### 長期施策の種（2年〜、構造改革レベル）
1. **インクルーシブ教育・特別支援充実**: 加配教員確保、合理的配慮の標準化
2. **不登校・多様な学びの場**: フリースクール認証、校内別室登校支援
3. **多文化共生子育て支援**: 外国ルーツ児童への日本語指導、保護者通訳
4. **教員働き方改革と部活動地域移行の本格運用**
