# 草川たくや｜まちづくり・経済アーカイブ
最終更新: 2026-04-25（議会会議録 平成30年〜令和7年 全20セッション抽出済）

## エグゼクティブサマリー

- **主要主張**（5本）
  1. リニア亀山駅＋鈴鹿亀山道路ICで「三重県全体の玄関口」化
  2. コストコ誘致＝市内経済活性化の象徴政策（場所固執を超えた戦略性）
  3. 地域公共交通の再設計（MONET/ライドシェア/事例集学習）
  4. Society 5.0×SDGs×まちづくり観光の重層戦略
  5. 産業構造転換期の好機を逃さない（航空宇宙・AIサーバー・自然エネルギー）

- **一貫性スコア**: A（H30初質問から「亀山の玄関口」「移住定住」「企業誘致」を一貫して柱に）
- **特徴**: ハードインフラ（道路・駅）・ソフト戦略（DX・観光）・産業誘致を統合的に論じる
- **未触れ重要領域**: 商店街振興の具体策、テレワーク移住、地域通貨、空き家活用等

## 3. 時系列進化（議会一般質問テーマから抽出）

### 2018-2019（初当選期・1期序盤）
- **リニア亀山駅＋移住定住**を初質問の柱に
- 企業誘致、工業団地、航空宇宙産業のクラスター形成
- 分散型エネルギー、地区防災計画

### 2020-2021（コロナ期）
- **Society 5.0×SDGs×観光**の重層戦略を本格展開
- 地域公共交通の再編・ライドシェア提案
- DXによるスマート自治体・新庁舎整備

### 2022-2023（1期終盤〜2期始動）
- 太岡寺町企業進出、地域まちづくり協議会
- リニア環境アセス、JR下庄駅周辺整備、鈴鹿川以南土地利用
- 公園緑地、生成AI業務導入

### 2024-2025（2期中盤〜現在）
- **コストコ誘致＝場所より戦略**（鈴亀道路IC周辺新提案）
- 大規模施設一体推進（新庁舎・新ごみ・コストコ）
- 公共交通の事例集学習（志摩・松阪・川越等を翻訳）
- 産業構造転換（AIサーバー、ソブリンAI拠点）
- 太陽光発電条例
- 関西本線電化・草津線直通化等の広域戦略

**進化の特徴**: 「点（リニア駅・コストコ）」→「面（IC周辺・産業構造転換）」→「動線（公共交通・広域連携）」と空間的に拡張。同時に「ハード→ソフト→DX」の重層化。

## 6. policy-synthesizer向けフィード（応援カード政策3本柱用）

### まちづくり・経済 — 推奨 短期/中期/長期 政策セット

#### 短期施策の種（〜6ヶ月、議会質問・運用改善レベル）
1. **コストコ誘致の市長トップセールス確認**: 鈴亀道路IC周辺戦略の進捗、産業振興奨励制度改正
2. **公共交通次期計画（R9〜）への市民意見反映**: MONET再設計の市民可視化、5自治体事例集学習結果
3. **太陽光発電適正導入条例の早期制定**: 議員提案 or 執行部案の選択
4. **AIサーバー・ソブリンAI拠点化に向けた誘致体制強化**: 産業立地ビジョン明示

#### 中期施策の種（6ヶ月〜2年、計画化レベル）
1. **関西本線電化・草津線直通化**: JR西日本・三重県との協議会設置
2. **昼生地区を含む地域公共交通新計画**: ライドシェア・乗合タクシー・路線バス三層
3. **鈴亀道路ICエリア土地利用計画**: コストコ＋関連商業集積
4. **空き家バンク条例＋移住促進**: テレワーク移住者向けインセンティブ
5. **観光DMO本格運用**: 鈴鹿川源流域・日本遺産・亀山茶の連携商品

#### 長期施策の種（2年〜、構造改革レベル）
1. **リニア亀山駅開業（2037年想定）に向けた都市再生**: 駅前再開発、複合施設、新交通結節
2. **航空宇宙＋AIサーバー＋環境エネルギー の三本柱産業構造**: クラスター形成計画
3. **コンパクトシティ・プラス・ネットワーク本格実装**: 中心市街地＋衛星集落の最適配置
4. **広域行政（三重県中部・鈴鹿〜亀山〜津）連携**: 共通課題の共同事業化
