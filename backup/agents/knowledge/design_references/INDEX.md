# デザイン参照ライブラリ INDEX

design-inspiration-researcher エージェントが収集した政治家広報物デザイン事例の蓄積場所。

## 使い方
- 各リサーチは日付フォルダに格納: `YYYY-MM-DD_テーマ名/`
- 各事例は番号フォルダに分離: `01_対象名/`、`02_対象名/`...
- 必ず `source_url.txt` と `analysis.md` を含める
- スクリーンショットは `reference.png` (or `.jpg`)
