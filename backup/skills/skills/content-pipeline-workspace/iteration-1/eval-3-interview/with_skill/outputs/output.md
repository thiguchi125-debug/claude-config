# 「準備が整ってから」は永遠に来ない――起業家・鈴木さんが教える、後悔しない決断の作り方

## はじめに

「いつか起業したい」と思いながら、何年も動けずにいる人はいないだろうか。

準備が足りない。スキルが足りない。お金が足りない。リスクが怖い。——そんな言い訳を並べているうちに、気づけば10年が過ぎている。

副業から始めて半年で本業の収入を超え、独立を果たした起業家・鈴木さんもかつてはそんな一人だった。会社員時代の閉塞感から抜け出すきっかけになったのは、一人の友人の、たった一つの問いだった。

## 「何が怖いの？」ではなく「10年後に後悔しないか？」

鈴木さんが行動できなかった理由は、失敗への恐れではなかったという。「何をしたいかわからなかった」のだ。

多くの人は「怖いから動けない」と思い込んでいるが、本当の問題は「目的地が見えていない」ことかもしれない。ゴールが不明確なまま走り出せないのは、ある意味で当然のことだ。

転機は友人の言葉だった。

「何が怖いの？」ではなく「10年後に後悔しないか？」

この問いは、鈴木さんの思考を根本から変えた。恐怖から逃げることを考えるのではなく、未来の自分からの視点で現在を見つめ直す。「10年後の自分に『なんであの時やらなかったんだ』と言われたくなかった」——その感覚が、行動への原動力になった。

## 副業から始めて、半年で独立へ

鈴木さんが選んだのは、いきなり会社を辞める無謀なチャレンジではなかった。まず副業から始め、本業の収入を超えたタイミングで独立という、リスクを最小化しながら前進する戦略だった。

この「出口確認してから飛ぶ」アプローチは、多くの起業志望者が見落としているポイントだ。リスクをゼロにすることはできないが、「確認できるリスク」と「取れるリスク」を見極めることはできる。

最初の1年は「本当にきつかった」と鈴木さんは振り返る。それでも「あの判断は正しかった」と言い切れるのは、自分が納得した基準で動いたからこそだろう。

## 8割の準備で走り出す

起業を考える人へのアドバイスとして、鈴木さんは2つの原則を語った。

**「準備が整ってから」は永遠に来ない**

これは多くの起業家が口を揃えて言うことだが、鈴木さんの言葉には重みがある。「8割の準備ができたら走り出す。残り2割は走りながら補う」——この感覚は、完璧主義者には受け入れがたいかもしれない。しかし現実には、100%の準備ができる前に市場は変わり、機会は消える。

**一人でやろうとしない**

もう一つは「得意な人に任せることの大切さ」。これは失敗して初めてわかったことだと鈴木さんは言う。すべてを自分でコントロールしたいという欲求は、起業家なら誰でも持つ。しかしそれが成長の足枷になることも多い。

「自分の得意でないことを抱え込む時間」は、「得意な人に任せて自分の強みに集中する時間」より、常にパフォーマンスが劣る。

## まとめ・アクション

鈴木さんのストーリーから学べることをまとめると：

1. **10年後の自分視点で決断する** — 「怖いから動けない」ではなく「10年後に後悔しないか」を問う
2. **副業から始めてリスクを管理する** — 全部賭けなくても、戦略的に前進できる
3. **8割準備で走り出す** — 完璧な準備を待つのをやめる
4. **得意な人に任せる** — 全部自分でやろうとしない

まず今日できることは、「10年後の自分に何と言われたくないか」を紙に書いてみることかもしれない。その答えが、あなたの次の一手を教えてくれるはずだ。

「準備が整ってから起業する」——この言葉が、夢を永遠に先送りにする最大の罠かもしれません。

先日、副業から始めて半年で本業の収入を超え、独立を実現した起業家・鈴木さんにインタビューする機会がありました。その言葉から、多くのビジネスパーソンが行動できない本当の理由が見えてきました。

転機になったのは友人の一言——「10年後に後悔しないか？」この問いは、恐怖から逃げる視点ではなく、未来の自分からの視点で現在を評価するフレームを与えてくれます。

第二に「一人でやろうとしない」こと。得意な人に任せることの重要性は、失敗して初めて実感したと鈴木さんは言います。自分の強みに集中し、それ以外は信頼できる人に委ねる——これはスタートアップでも大企業でも変わらない普遍の原則です。
