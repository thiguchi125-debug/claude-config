草川たくや名義のブログ記事（深掘り版・ノーマル版とも）は、**記事本文の1行目を必ず以下の定型挨拶で開始**する。

- `こんにちは、亀山市議会議員の草川たくやです。`
- 時間帯に応じて `こんばんは、亀山市議会議員の草川たくやです。` も可

**Why**: ユーザー本人の発信スタイルとして定着している自己紹介。これがないと「草川たくやらしさ」が損なわれ、読者にも違和感を与える。2026-04-23 のセッションで、和賀白川線記事をセリフ引用から書き始めたところユーザーから明確に修正指示があった。

**How to apply**:
- blog-writer.md / blog-writer-normal.md / voice-dna.md §1 / §6 / §9 すべて更新済み（2026-04-23）
- 名乗りの直後に1行空け、市民のセリフ・具体エピソード・問いかけなど従来の冒頭フックへ展開する（フック自体は引き続き必須）
- LINE 媒体も同様に「亀山市議会議員の」を含めた完全形に統一
- Facebook は従来どおり定型挨拶なし（直接本題に入る）
