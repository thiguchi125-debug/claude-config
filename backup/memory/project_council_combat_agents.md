2026-04-26新設。議員業務カバレッジ分析で「議場戦闘力」と「動画戦略」が穴と判明し、最優先3本を一括投入。

**Why**：既存18本は発信・政策形成は手厚いが、議場での戦闘力（採決判断・答弁準備）と20-40代向け動画リーチが薄かった。来年10月選挙までに必要な装備として最優先3本を補強。

**How to apply**：
- **agenda-analyzer**：議案分析・採決判断・補正予算/条例改正/請願陳情カルテ。8軸評価＋賛否両論併記＋voice-dna/compass整合性チェック。複数議案の一括トリアージモードあり
- **counter-argument-simulator**：議場リアルタイム戦闘マニュアル。想定答弁3パターン(BEST/STANDARD/EVASIVE)＋5層階段カード(受け止め→事実確認→他自治体比較→本質回帰→決断要求)＋撤退ライン。policy-validatorが offline-defensive、こちらは live-floor combat
- **video-content-strategist**：YouTube/Shorts/TikTok/Reels台本＋議会切り抜き候補抽出（5基準）＋サムネ/タイトル/SEO/アンチ想定。3秒で掴む・30秒で響かせる・60秒で動かす

**棲み分け**：
- council-material-creator＝攻めの構成、counter-argument-simulator＝守りと反撃
- policy-validator＝政策単体検証、agenda-analyzer＝議案単位の判断材料
- sns-content-creator＝静止画文字主体、video-content-strategist＝動画専任

**並列起動**：3本とも kameyama-researcher / policy-archive-miner と並列前提。

**未着手の優先候補**（次フェーズ）：
1. formal-petition-writer（公式要望書ドラフト）
2. electoral-district-strategist（選挙地区戦略・票分析）
3. press-response-handler（記者対応・メディア応答）
4. crisis-communicator / budget-reviewer / supporter-network-strategist（中長期）
