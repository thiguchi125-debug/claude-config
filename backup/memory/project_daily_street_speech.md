2026-04-26新設。草川たくや（亀山市議会議員）の駅前・街頭での20分演説 talking points を毎日その日の素材で生成する専任エージェント。

**Why**: speech-writer（式典・長尺）でもcommunity-rally-speaker（地区集会）でもカバーできない「日々の駅頭・街角での20分ループ演説」を専門化する必要があった。鮮度（今日のニュース）＋ローカル素材（直近nichijo・市民意見）＋政策軸（コンパス3軸）を毎日その場で組み立てる。

**How to apply**:
- 起動トリガー：「今日の街頭演説」「街頭演説の要旨」「駅前で話す」「街宣ネタ」「今日の街宣」「駅頭スピーチ」等
- 出力仕様（草川指定）：
  - 長さ：20分（4〜5分ループ × 4周で構成、繰り返しOK前提）
  - 形式：原稿不要、talking points outline のみ
  - パッケージ：3本必須（A重め=政策深掘り／B親しみ=共感ストーリー／C攻め=立場明確化）
- 自動収集ソース：
  1. 🧭 政策コンパス（policy_compass.md・最重要）
  2. 📰 ニュースDB（f2eefc66...）今日分
  3. 📔 nichijo直近1週間（292c_c関連）
  4. 📝 市民意見リスト直近2週間
  5. 🎯 政策候補DB（ds:6f1895ac...）推進中ステータス
  6. voice-dna.md
- 構造（1ループ）：掴み30秒→軸接続1分→直近活動/市民の声1分→政策具体1分→応援部魂結び1分
- 公職選挙法配慮：他候補名指し批判禁止、「投票して」型禁止、政策論争として展開
- 結びに必ず「応援部魂」を埋め込む（パッケージごとに表現可変）
- compass未確定でも暫定動作可（archive-miner themes 3本を直接参照）
- 関連エージェント：speech-writer（フル原稿希望時に展開）、community-rally-speaker（地区集会）
