AIインタビュー（https://depth-interview-kusagawa.vercel.app/ 配下）を紹介・案内するSNS投稿（4PF=Threads/X/Instagram/Facebook）には、以下の定型文を**必ず一字一句正確に**含める。

```
※本システムは安野貴博氏（参院議員/AIエンジニア）が開発したdepth interviewのプログラムを元にしたプロトタイプの試験的な利用になります。有賀啓介氏（コンサルタント）のご支援で実施しています。
※個人情報や機密情報の入力は禁止です。
```

**Why:** 草川本人指示（2026-05-01）。「今後もそのようにして」と明言。安野貴博氏（参院議員/AIエンジニア）と有賀啓介氏（コンサルタント）への正しいクレジット表示は協力者への信義として必須。個人情報禁止の注意は誤入力リスク回避のため。

**How to apply:**
- AIインタビュー紹介投稿全般（4PF全部）に同一文言を含める
- 短文プラットフォーム（X 140字）でフィットしない場合は、**スレッドの2投目**として独立配置する。本文側を削るより thread 化を優先
- ai-interview-sns-poster エージェント仕様にも反映（既存 boilerplate footer の文言を本ルールの文言に置換）
- 安野貴博氏の肩書は **「参院議員/AIエンジニア」**（順序固定、スラッシュ区切り）
- 有賀啓介氏の肩書は **「コンサルタント」**
- ※マークは半角ではなく全角「※」、間に半角スペースなし

**反映漏れ検出:** 投稿前チェックで、4PF全てに「安野貴博」「有賀啓介」「個人情報」3キーワードが含まれることを確認する。
