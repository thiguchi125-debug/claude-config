## 訪問完了時の運用フロー(自動同期版)

1. その日のタスクを開く(例: 4/28(火)坂下偵察+市瀬東端)
2. プロパティに数値入力:
   - 訪問軒数(在宅): 在宅対面+在宅拒否の合計
   - 訪問軒数(投函): 不在投函の数
3. ステータスを Done に
4. → 📊進捗DBの「坂下」レコードの「在宅軒数_自動」「投函軒数_自動」「訪問済_自動」「達成率_自動」が自動更新
5. 効率(軒/h)はタスクの「要約」or 進捗DBの「実効効率(軒/h)」に手動入力
